## ❓ FAQ

**Q: Do I need the API key for basic features?**
A: No! Only for advanced features (6 new ones).
Basic features (Resume, Voice/Video Interviews, Quiz) work without it.

**Q: Is my API key safe?**
A: Yes! Stored locally on your computer only.
Never sent to servers or logged.

**Q: How many requests can I make?**
A: 1M tokens/day, 15 requests/minute.
More than enough for unlimited interview prep.

**Q: Can I use it offline?**
A: Advanced features need internet (for Gemini).
Basic features work offline after first load.

**Q: Will my data be saved?**
A: All data saved locally on your computer.
No cloud storage, no tracking.

**Q: Can I export my progress?**
A: Yes! PDF exports include all charts + metrics.

**Q: What if I lose my API key?**
A: Generate a new one at aistudio.google.com
Old keys can be revoked instantly.
