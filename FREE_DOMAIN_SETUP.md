# 🎯 FREE DOMAIN SETUP GUIDE (Freenom)

Your app is **100% free** - including the domain! Follow these simple steps:

## ✅ Step 1: Register FREE Domain

1. Go to **https://www.freenom.com**
2. Sign up (free account)
3. Search for: **nexhire.tk** (or nexhire.ml, nexhire.ga)
4. Click "Get It Now" → Checkout (FREE)
5. Select period: **12 Months FREE**
6. Complete registration

## ✅ Step 2: Point Domain to GitHub Pages

1. In Freenom dashboard, click **"My Domains"**
2. Select **nexhire.tk** → **Manage Domain**
3. Click **"Manage Freenom DNS"**
4. Delete existing records and add:

```
Name: @
Type: A
TTL: 3600
Target: 185.199.108.153

Name: @
Type: A
TTL: 3600
Target: 185.199.109.153

Name: @
Type: A
TTL: 3600
Target: 185.199.110.153

Name: @
Type: A
TTL: 3600
Target: 185.199.111.153

Name: www
Type: CNAME
TTL: 3600
Target: manikanta-devs.github.io
```

5. Save changes (wait 15-30 mins for DNS to propagate)

## ✅ Step 3: Enable GitHub Pages Custom Domain

1. Go to **https://github.com/manikanta-devs/interview-confidence-hub**
2. Settings → Pages
3. Under "Custom domain": Enter **nexhire.tk**
4. Click Save
5. Check **"Enforce HTTPS"** (when it appears)

## ✅ Done! 🎉

Your app is now live at:
```
https://nexhire.tk
```

**Completely FREE forever!**
